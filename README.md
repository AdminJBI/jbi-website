# Joint & Bone Institute — Website Design

Squarespace build for **Joint & Bone Institute of California** (JBI), an orthopedic surgery practice serving Bakersfield, Delano, and Potterville.

## Files

| File | Purpose |
|---|---|
| `index.html` | Reference design (full visual spec for Direction A "Deep Ocean") |
| `SQUARESPACE-BUILD.md` | **Active build playbook** — current state, all code, todos |
| `FRAMER-BUILD.md` | Alternative Framer path (not used — kept for reference) |
| `preview-homepage.html` | Earlier iteration |
| `logo/` | Logo SVGs, PNGs, favicon |

## Current state (snapshot)

- **Header**: Squarespace native with custom mega-menu via Code Injection
- **Hero**: Video background + native blocks
- **Stats**: Hardcoded Code Block (4 stats)
- **Team**: Hardcoded Code Block (3 surgeons)
- **Specialties CMS**: Squarespace Blog page at `/specialties` with 11 of 18 posts created
- **Mega-menu**: 3-column dropdown (Hip & Knee / Sports & Regenerative / Hand & Fracture)
